# Python 期末复习闯关小游戏网页

这是一个适合程序设计基础课程展示的 Python 网页小游戏。项目使用 Streamlit 开发，只需要 Python 代码就能生成可交互网页。

## 在线访问

本游戏已部署到 Streamlit Community Cloud，电脑、手机浏览器都能直接打开使用，无需安装任何 Python 环境。

🔗 在线地址：https://python-quiz-app-dsp35cudw2tzymqcsnym7w.streamlit.app

## 选题名称

Python 期末复习闯关：从基础语法到排序、递归与二维列表

## 项目亮点

- 题目取材自本课程实验报告、随堂作业、课后作业和期末复习重点。
- 网页端可选择简单、普通、挑战三种难度。
- 系统随机抽题，避免每次展示完全相同。
- 自动判断答案、计算得分、显示称号。
- 提交后展示每道题解析，方便复习。
- 按知识点生成得分柱状图，显示薄弱环节。
- 题库查阅版整理在 `题库.md`，网页运行版写在 `app.py` 的 `QUESTION_BANK` 中。

## 题库说明

当前题库共 110 题：

- 简单模式：25 题
- 普通模式：40 题
- 挑战模式：45 题

题型包括：

- 概念选择题
- 代码输出题
- 代码填空题
- 代码阅读题

主要知识点包括：

- 变量与赋值、变量命名、注释
- 运算符：整除 `//`、取余 `%`、乘方 `**`
- 字符串：下标、切片、字符串方法、格式化
- 流程控制：`if` / `elif` / `else`、`for`、`while`、`range`
- 列表、元组、字典、集合
- 列表方法：`append`、`sort`、`remove` 等
- 函数：定义、返回值、参数传递、可变参数
- 递归：递归求和、递归判断素数等
- 二维列表：矩阵转置、对角线、初始化陷阱
- 排序：选择排序、冒泡排序
- 数论算法：最大公约数 GCD、最小公倍数 LCM
- 文件读取与异常处理：`open`、`read`、`try` / `except`
- 部分拓展题：列表推导式、`lambda`、`filter`

`题库.md` 是题库查阅版，和当前 `app.py` 的题库内容保持一致。正式改题时建议优先修改 `app.py`，再同步更新 `题库.md`。

## 10分钟展示流程

1. 第1分钟：介绍选题背景。期末复习知识点多，所以做一个互动闯关工具。
2. 第2分钟：说明技术路线。使用 Python + Streamlit，把 Python 程序变成网页。
3. 第3-5分钟：现场演示。选择难度、答题、提交、查看分数和解析。
4. 第6-7分钟：讲解核心代码。题库用列表和字典保存，函数负责抽题和评分。
5. 第8分钟：说明题库设计。按难度、知识点和题型组织题目。
6. 第9分钟：总结课程知识点。变量、字符串、列表字典、函数、二维列表、排序、文件异常。
7. 第10分钟：项目反思和改进。后续可加入专项练习、错题本、排行榜。
